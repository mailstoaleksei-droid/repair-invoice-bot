## 🔧 Ключевые улучшения v7.0

### 1. 🔴 Критическое исправление: NETTO vs BRUTTO

**Проблема в v4.0 и v6.0:**
```python
# ❌ НЕПРАВИЛЬНО: Всегда использовал BRUTTO
data['total_price'] = gesamt  # Всегда Gesamt (BRUTTO)
```

**Решение в v7.0:**
```python
# ✅ ПРАВИЛЬНО: Определяет тип счёта
if is_internal:
    data['total_price'] = gesamt   # BRUTTO для Internal
else:
    data['total_price'] = netto    # NETTO для External
```

**Влияние:**
- v4.0/v6.0: Неправильные суммы для External счетов Auto Compass
- v7.0: Правильные суммы для всех счетов
