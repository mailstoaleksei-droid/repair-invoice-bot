### Ошибка: "Module not found: unified_telegram"
**Решение:** Скопировать unified_telegram.py в ту же папку

### Ошибка: "Excel file not found"
**Решение:** Проверить путь к Repair_2025.xlsx

### Ошибка: "Permission denied"
**Решение:** Закрыть Excel файл перед запуском

### Программа зависла
**Решение:** 
1. Нажать Ctrl+C
2. Проверить логи в папке logs/
3. Перезапустить
