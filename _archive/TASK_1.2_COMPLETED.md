## 🎯 КЛЮЧЕВЫЕ ВОЗМОЖНОСТИ

### Throttling
```python
# Не более 1 сообщения каждые 2 секунды
notifier = TelegramNotifier(bot, chat_id, throttle_interval=2.0)
```

**Преимущества:**
- Предотвращает блокировку бота Telegram
- Не перегружает пользователя уведомлениями
- Автоматические задержки между сообщениями

### Детальная информация
Каждое уведомление включает:
- ✅ Имя файла
- ✅ Поставщик
- ✅ Номер счета
- ✅ Дата счета
- ✅ Сумма
- ✅ Номер машины
- ✅ Строка Excel (для успешных)
- ✅ Причина проблемы (для manual)

### Форматирование
- **HTML разметка** для читаемости
- **Emoji** для быстрой визуальной идентификации
- **Code blocks** для имен файлов
- **Жирный текст** для важной информации

## 🔍 TROUBLESHOOTING

### Уведомления не приходят

**Проверьте:**
1. `TELEGRAM_ENABLED = True` в `process_pdf_v3.py`?
2. `DETAILED_NOTIFICATIONS = True`?
3. Токен бота правильный?
4. Chat ID правильный?

**Отладка:**
```python
# В process_pdf_v3.py после импортов
if TELEGRAM_ENABLED:
    print(f"✓ Telegram включен, Chat ID: {TELEGRAM_CHAT_ID}")
else:
    print("✗ Telegram выключен")
```

### Слишком много уведомлений

**Решение 1:** Увеличить throttle
```python
notifier = TelegramNotifier(bot, TELEGRAM_CHAT_ID, throttle_interval=3.0)
```

**Решение 2:** Отключить начальные уведомления
```python
# Оставить только успех/дубликат/manual
```

**Решение 3:** Только финальная сводка
```python
DETAILED_NOTIFICATIONS = False
```

### Ошибка "Module not found: telegram_notifications"

**Причина:** Файл `telegram_notifications.py` не в той же папке

**Решение:**
```bash
# Проверьте что оба файла в одной папке:
PDF_Processor/
├── telegram_notifications.py  ← должен быть здесь
└── process_pdf_v3.py           ← и здесь
```
