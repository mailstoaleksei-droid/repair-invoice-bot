# 🚀 БЫСТРЫЙ СТАРТ - Задача 1.1

## ⚡ Установка за 3 минуты

### 1️⃣ Установить зависимости
```bash
pip install -r requirements.txt
```

Или вручную:
```bash
pip install watchdog pyTelegramBotAPI pdfplumber openpyxl
```

### 2️⃣ Скопировать файлы

Скопируйте в папку `PDF_Processor`:
- ✅ `file_monitor.py`
- ✅ `telegram_bot_v3.py`

### 3️⃣ Запустить бота
```bash
cd PDF_Processor
python telegram_bot_v3.py
```

### 4️⃣ В Telegram
1. Отправьте `/start`
2. Нажмите **🔍 Мониторинг**
3. Нажмите **▶️ Запустить мониторинг**

✅ **ГОТОВО!** Добавьте PDF в папку `EingangsRG` и получите уведомление через 5 секунд!

## ❗ ВАЖНО

### Paths
Убедитесь что пути правильные в `telegram_bot_v3.py`:
```python
PDF_FOLDER = r"C:\...\EingangsRG"      # ← проверьте
PROCESSED_FOLDER = r"C:\...\RG 2025..."  # ← проверьте
MANUAL_FOLDER = r"C:\...\manual"       # ← проверьте
```

### Токен бота
В `telegram_bot_v3.py` должен быть ваш токен:
```python
TELEGRAM_BOT_TOKEN = "ВАШ_ТОКЕН"
TELEGRAM_CHAT_ID = "ВАШ_ID"
```
