# 🔧 ИСПРАВЛЕНИЕ ОШИБКИ КОДИРОВКИ

## ❌ Проблема:

При запуске обработки возникла ошибка:
```
UnicodeDecodeError: 'utf-8' codec can't decode bytes in position 245-246
```

**Причина:** Один или несколько PDF файлов содержат нестандартную кодировку, которую `pdfplumber` не может прочитать стандартным способом.

---

## ✅ РЕШЕНИЕ: Используйте обновлённую версию v7.2

### Что исправлено в v7.2:

1. ✅ **Обработка ошибок кодировки** на уровне страницы
   - Если страница не читается с UTF-8, пробуем без layout
   - Если и это не работает, пропускаем страницу

2. ✅ **Обработка критических ошибок** на уровне файла
   - Если файл вообще не открывается, перемещаем в manual
   - Отправляем уведомление в Telegram с причиной

3. ✅ **Продолжение обработки** остальных файлов
   - Один проблемный файл не останавливает всю обработку
