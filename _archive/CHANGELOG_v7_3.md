## ✨ НОВЫЕ ФУНКЦИИ

### `extract_all_trucks_from_text(text)`
Извлекает ВСЕ номера машин из текста PDF
- Поддержка: GR-OO, HH-AG, DE-FN
- Возвращает список уникальных номеров
- Используется для разделения множественных машин

### `detect_gutschrift(text)`
Определяет является ли документ Gutschrift (возврат)
- Ищет "GUTSCHRIFT" или "Gutschrift" в тексте
- Возвращает True/False
- Используется во ВСЕХ функциях extract

### `extract_netto_prioritized(text)`
Приоритетное извлечение NETTO суммы
- Приоритет 1: Nettobetrag
- Приоритет 2: Nettosumme
- Приоритет 3: Netto
- Приоритет 4: Summe netto
- Приоритет 5: Zwischensumme netto
- Последний вариант: Gesamt/Gesamtbetrag
