## ⚡ Установка за 2 минуты

### 1️⃣ Скопировать файлы

Скопируйте в папку `PDF_Processor`:
- ✅ `telegram_notifications.py`
- ✅ `process_pdf_v3.py`

### 2️⃣ Обновить telegram_bot_v3.py

Откройте `telegram_bot_v3.py` и измените строку:

```python
# Найдите эту строку (около строки 24):
SCRIPT_PATH = r"...\process_pdf_v2.py"

# Измените на:
SCRIPT_PATH = r"...\process_pdf_v3.py"
```

### 3️⃣ Перезапустить бота

```bash
# 1. Остановить (Ctrl+C)
# 2. Запустить заново
python telegram_bot_v3.py
```

### 4️⃣ Тест!

**В Telegram:**
1. Нажмите **🔄 Обработать PDF**
2. Получите детальные уведомления!

✅ **ГОТОВО!**
