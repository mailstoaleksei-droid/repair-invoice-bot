# Roadmap To 90%+ — 2026-03-27

## Цель

Поднять корректную автоматическую обработку счетов до `90%+` через безопасный цикл улучшений на текущем наборе PDF.

## Рабочая схема

1. Берем текущие счета из `EingangsRG`
2. Обрабатываем их в режиме `report_only`
3. PDF не перемещаются
4. Рабочий Excel не меняется
5. Создается отдельный отчет по запуску
6. Проверяем ошибки по поставщикам и полям
7. Исправляем алгоритм
8. Повторяем цикл до стабильного качества
9. Только потом включаем production-режим

## Фаза 1. Безопасный контур

- [x] Включить режим без перемещения файлов
- [x] Отключить запись в рабочий Excel

## Фаза 4. Итеративный цикл

Рабочий процесс:

1. Вы запускаете обработку
2. Я анализирую отчет
3. Вы отмечаете ошибки по фирмам

5. Снова запускаем на тех же счетах
6. Сравниваем прогресс

Промежуточные KPI:
- [ ] `50%+` корректной обработки
